# geometry-volume-app
Simulate a simple geometry volume calculator

# Project Structure

.
 geometry/            # Logic for geometric calculations
|   |-- __init__.py
|   |-- box.py
|   |-- cone.py
|   |-- cylinder.py
|   |-- sphere.py
|-- tests/               # Unit tests
|   |-- __init__.py
|   |-- test_box.py
|   |-- test_cone.py
|   |-- test_cylinder.py
|   |-- test_sphere.py
|-- main.py              # Entry point of the application
|-- README.md            # Project documentation
|-- requirements.txt     # List of dependencies

# How to Run main.py

To execute the main application run:

- python main.py

# How to Run Tests

This project uses pytest for automated testing. To run all tests use:

- pytest

# Dependencies

- python 3.x
- pytest 9.x
